# Cyber-Climb
Cyber Climb is a hilarious, high-energy reimagining of the classic Snakes &amp; Ladders board game. Designed for 2 to 4 local players, it takes the traditional game and injects it with modern mechanics like an interactive 3D tumbling dice, dynamic visual themes (including Cyber Neon and Emoji Madness), and strategic buffs like Shields and Extra Spins.
# 🐍 Cyber Climb: Modern Snakes & Ladders

A completely reimagined, highly interactive, modern web-based version of the classic Snakes & Ladders game. Built entirely with Vanilla JavaScript, HTML5, and Tailwind CSS in a clean, single-file architecture.

## ✨ Features

* **3D Physics Dice:** A fully interactive 3D CSS cube that physically pops out, tumbles, and snaps to the accurate rolled number.
* **Dynamic Theme Engine:** Switch instantly between 🌐 **Cyber Neon**, 🪵 **Classic Wood**, and 🤪 **Emoji Madness** themes. 
* **Procedural SVG Graphics:** Snakes and Ladders are mathematically generated using precise SVG Bezier curves and line geometry, ensuring they scale flawlessly on any resolution.
* **Buff System:**
  * 🛡️ **Shields:** Pick up a shield to protect yourself from your next snake bite.
  * ⚡ **Extra Spin:** Land on lightning to immediately roll again.
* **"Emotional Damage" System:** Get bitten by a snake without a shield? Prepare to be hit with a randomized, screen-shaking internet meme popup (e.g., "SKILL ISSUE", "OOF SIZE: MEGA").
* **Custom Multiplayer:** Supports 2 to 4 local players. Input custom names, and watch the smart tokens auto-arrange themselves if multiple people land on the same square.
* **Step-by-Step Animation:** Tokens physically hop square-by-square instead of just teleporting across the board.

## 🚀 How to Play

Because this is a zero-dependency, single-file application, playing is incredibly easy!

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Select your Theme, Player Count, and enter names.
4. Click **PLAY GAME**.
5. Click the 3D dice to roll! (Remember: You must roll a 1 or a 6 to unlock your token and start moving).

## 🛠️ Tech Stack

* **HTML5** (Structure and SVG Canvas)
* **Tailwind CSS** (via CDN for rapid responsive layout)
* **Vanilla JavaScript** (Game logic, 3D math, path generation, DOM manipulation)
* *No external image assets, libraries, or dependencies used!*
